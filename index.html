Highlighted Projects

ASIC Design Engineer — RISC-V CPU

Designed a 3-stage pipelined RISC-V CPU with CSR support in Verilog, verified with SystemVerilog testbenches. Integrated cache memory and achieved low-latency performance in Skywater 130nm using Cadence Innovus.

PCB Design Engineer — Custom MIDI Keyboard

Built a custom multi-layer PCB for a USB MIDI keyboard using KiCad. Implemented grounding, power integrity, and signal testing using oscilloscope debugging and circuit analysis.

Control System Engineer — Visual Servoing & Grasping Control

Developed PD-based joint-space and torque controllers for a 7-DOF Sawyer arm to track AR tags, tuning controller gains to reduce trajectory error and motion jitter. Designed IK solver and grasp synthesis for Allegro Hand in MuJoCo using Levenberg–Marquardt optimization and force closure metrics.

Process Integration Engineer — Microfabrication Technology

Fabricated NMOS devices from bare silicon wafers over a 12-week cleanroom process flow. Performed photolithography, gate oxidation, ion implantation, contact etch, and metal deposition. Characterized resistors, capacitors, MOS capacitors, and long-channel NMOS transistors using 4155B semiconductor analyzer and 4284A LCR meter for I–V and C–V measurements.

Robotics System Developer — TurtleBot Path Planning & State Estimation

Built RRT, sinusoidal, and Dubins path planners for a unicycle-model TurtleBot. Implemented Lyapunov-based tracking controller in ROS and applied Kalman Filter (KF) and Extended Kalman Filter (EKF) for state estimation, comparing MSE vs. dead reckoning for navigation accuracy.

Semiconductor Device Designer — N-MOSFET Design Project

Utilized Synopsys Sentaurus device simulation software to design an N-channel Si MOSFET with gate length LG = 25 nm (relevant for the 40 nm generation of CMOS technology). Met specified performance requirements within constraints using Sentaurus process and device simulation modules.

Robotics Software Engineer — TurtleBot Maze Solver

Developed a Python-based ROS system for TurtleBot 2.0 to autonomously navigate arbitrary mazes. Implemented mapping, layout storage, and traversal logic, integrating ROS topics and services for real-time navigation.

CPU Designer — CS61CPU Project

Developed a processor capable of executing RISC-V machine code with instruction fetch, decode, execute, memory access, and write-back stages. Integrated pipelining to enhance throughput, verified functionality with testbench-based simulation, and analyzed performance metrics.
